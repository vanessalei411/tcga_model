TCGA Breast Cancer Gene Expression Visualizer

This project began with a simple question: can software uncover patterns in clinical research that are difficult to detect through traditional analysis?

By organizing TCGA gene expression data into an interactive visualization platform, the project reveals relationships between gene expression and breast cancer subtypes, enabling more intuitive exploration of clinically relevant patterns. My TCGA visualizer takes gene expression data from 151 real patients across 54,647 genes. Built with a combination of React, Flask, and Machine Learning classifier.
